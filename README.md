Sitio web educativo desarrollado para una exposición universitaria sobre los Derechos Reales.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript

## Contenido

- Concepto de Derechos Reales

- Derecho de Propiedad
- Relaciones de Poder
- Posesión y Tenencia
